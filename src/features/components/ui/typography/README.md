# Typography Primitive

Base typography component for Thuocare.

## Usage

```tsx
import { Typography } from "./typography";

<Typography variant="headline-lg">Hello World</Typography>;
```

## Props

- `variant`: Typography variant (display, headline, title, body, label)
- `color`: Text color
- `weight`: Font weight (normal, medium, semi-bold, bold)
- `align`: Text alignment (left, center, right, justify)
- Standard `TextProps`
