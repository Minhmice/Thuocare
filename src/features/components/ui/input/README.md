# Input Primitive

Base text input component for Thuocare.

## Usage

```tsx
import { Input } from "./input";

<Input label="Email" placeholder="Enter your email" />;
```

## Props

- `label`: Input label
- `error`: Error message or boolean
- `leftIcon`: Icon displayed on the left
- `rightIcon`: Icon displayed on the right
- `containerStyle`: View style for the container
- `inputStyle`: Text style for the input
- Standard `TextInputProps`
