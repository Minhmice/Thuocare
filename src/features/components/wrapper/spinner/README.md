# Spinner Wrapper

App-themed spinner for loading states.

## Usage

```tsx
import { Spinner } from '../wrapper/spinner';

<Spinner size="lg" center />
```

## Props

- `size`: 'sm', 'md', 'lg' or a number
- `color`: Override color
- `center`: Wrap in a centered `View` with `flex: 1`
- `style`: Container view style
